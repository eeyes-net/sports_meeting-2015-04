# 西安交通大学第53届校运动会官方网站

<http://2015.eeyes.net/> （已下线）

## 时间轴

* 2015年03月22日：首次上线测试
* 2015年04月09日：正式上线测试
* 2015年04月24日、25日：运动会期间，累计两千多次访问量
* 2016年03月26日：网站正式下线
* 2017年05月29日：开始整理代码

## 环境要求

* `php >= 5.3`
* 打开`PDO_mysql`拓展
* 其他具体要求请参考<http://www.kancloud.cn/manual/thinkphp/1681>
